📝 Testing Instructions:
  1. Click the extension icon and add this page to allowed URLs
  2. Open browser console (F12) to see debug messages
  3. Type in the fields below and wait 1 second
  4. Fields with errors will get a red border on the bottom
  5. Click on the field to see all errors
  6. Click any green button to fix that error!
  7. Double-click any word to get synonyms!
